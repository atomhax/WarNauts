Ideas:
Chronowarpers

  - Moves:
  - Move that causes all Chronomorph units to Teleport

Nedrex

 - Moves:

Nameless Infestation Race
  - Ability to place infested versions of other race
  
Nameless Synergy Race
  - Tesla tower units (Creates electricity between two units)
